discussion_answers

Description

For each course's discussion forums, contains the list of the answers that users submitted to discussion questions.

Columns

NameDescription
discussion_answer_idThe id of the answer in the discussion forum.
ualberta_discussions_user_idEncrypted Coursera user id for ualberta discussions data.
course_id22 character unique string identifying an individual course
discussion_answer_contentContent of the answer (CML)
discussion_question_idThe id of a question in the discussion forum.
discussion_answer_parent_discussion_answer_idParent answer if this is a nested reply (we cap nesting at 1 level)
discussion_answer_created_tsThe timestamp of when the discussion answer was created.
discussion_answer_updated_tsThe timestamp of when the discussion answer was updated.

SQL create statement

CREATE TABLE discussion_answers (
    discussion_answer_id varchar 
    ,ualberta_discussions_user_id VARCHAR(50) NOT NULL
    ,course_id varchar 
    ,discussion_answer_content varchar 
    ,discussion_question_id varchar 
    ,discussion_answer_parent_discussion_answer_id varchar 
    ,discussion_answer_created_ts timestamp 
    ,discussion_answer_updated_ts timestamp 
    ,PRIMARY KEY (discussion_answer_id)
    ,FOREIGN KEY (discussion_answer_parent_discussion_answer_id) REFERENCES discussion_answers(discussion_answer_id)
);
