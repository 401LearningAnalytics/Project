programming_assignment_submission_schema_parts

Description

Information of individual parts of a programming assignment

Columns

NameDescription
programming_assignment_idAssignment that this programming schema part lives in.
programming_assignment_submission_schema_part_idThis part's id within the assignment.
programming_assignment_submission_schema_part_titleName of the submission.
programming_assignment_submission_schema_part_typeThe type ( "grid", "numeric", "regex")
programming_assignment_submission_schema_part_orderThe order within the assignment (lowest comes first).
programming_assignment_submission_schema_part_max_scoreMax score of the part.
programming_assignment_submission_schema_part_is_optionalWhether this part is optional or not.
programming_assignment_submission_schema_part_xacgt8The internal authoring version of the part, only used by authoring tools. Name shortened from 'programming_assignment_submission_schema_part_authoring_version' for compatibility.
programming_assignment_submission_schema_default_g663i6The default incorrect feedback of a submission. Name shortened from 'programming_assignment_submission_schema_default_incorrect_feedback' for compatibility.

SQL create statement

CREATE TABLE programming_assignment_submission_schema_parts (
    programming_assignment_id VARCHAR(50) 
    ,programming_assignment_submission_schema_part_id VARCHAR(50) 
    ,programming_assignment_submission_schema_part_title VARCHAR(65535) 
    ,programming_assignment_submission_schema_part_type VARCHAR(50) 
    ,programming_assignment_submission_schema_part_order INT4 
    ,programming_assignment_submission_schema_part_max_score INT4 
    ,programming_assignment_submission_schema_part_is_optional BOOL 
    ,programming_assignment_submission_schema_part_xacgt8 INT4 
    ,programming_assignment_submission_schema_default_g663i6 VARCHAR(65535) 
    ,PRIMARY KEY (programming_assignment_submission_schema_part_id, programming_assignment_id)
    ,FOREIGN KEY (programming_assignment_id) REFERENCES programming_assignments(programming_assignment_id)
);
