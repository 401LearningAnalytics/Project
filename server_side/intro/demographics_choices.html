demographics_choices

Description

Stores choices to the questions from the demographics survey and user intent questions.

Columns

NameDescription
question_idInteger id for the questions
choice_idThe choice id helps to identify how a learner responded to a demographic question. The choice id itself is not a unique identifier, but the question id/choice id combo is unique and relates to a specific choice description (choice_desc) that provides the text of a learner's answer selection.
choice_descIndicated by a unique question id/choice id combo, this provides the text of a learner's answer selection to a demographic question.

SQL create statement

CREATE TABLE demographics_choices (
    question_id int4 
    ,choice_id int4 
    ,choice_desc varchar 
    ,PRIMARY KEY (question_id, choice_id)
);
