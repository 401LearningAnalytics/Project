demographics_question_types

Description

Stores question types for the demographics questions.

Columns

NameDescription
question_type_idIndicates the type of answer required by the question: 0 (mulitple choice), 1 (integer), 2 (text).
question_type_descDescribed the answer required by each question type: 0 (mulitple choice), 1 (integer), 2 (text).

SQL create statement

CREATE TABLE demographics_question_types (
    question_type_id INT4 
    ,question_type_desc VARCHAR(25) 
    ,PRIMARY KEY (question_type_id)
);
