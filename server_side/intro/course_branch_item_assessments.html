course_branch_item_assessments

Description

For each course branch, provide the mapping between course branches and versioned assessments.

Columns

NameDescription
course_branch_idEach session is associated with a single branch. Learners enrolled in a session see the course contents associated with this branch.
course_item_id5 character alphanumeric string identifying an individual item within a course. Items that have the same id that are in different branches of the same course are considered to be the same for the purposes of progress and grade computations. For example, if you complete item xxxxx in branch A, then you have completed it in branch B even if item xxxxx in branch B is very different from item xxxxx in branch A.
assessment_idEach quiz is assigned a unique id that is a combination of a base id and version.

SQL create statement

CREATE TABLE course_branch_item_assessments (
    course_branch_id varchar 
    ,course_item_id varchar 
    ,assessment_id varchar 
);
