course_branch_grades

Description

This table provides the grading event of when the user reached his or her highest grade in one course branch. Also provides the passing/not-passing state for each user and course branch.

Columns

NameDescription
course_branch_idEach session is associated with a single branch. Learners enrolled in a session see the course contents associated with this branch.
ualberta_user_idEncrypted Coursera user id for ualberta data.
course_branch_grade_tsThe timestamp for when a learner's branch grade has changed. This occurs everytime a grading event takes place.
course_passing_state_idA learner can be in one of four "states" with respect to passing a branch: 0 means they have started the branch but not passed, 1 means they have completed the branch without identity verification, 2 means they've completed the branch with identity verification, 3 means the branch is not passable.
course_branch_grade_overall_passed_itemsThe number of items in the branch that the learner has passed.
course_branch_grade_overallThe grade a learner has in the branch, according to the instructor-set grading policy, based on passed items.
course_branch_grade_verified_passed_itemsThe number of items in the branch that the learner has passed and verified their identity.
course_branch_grade_verifiedThe grade a learner has in the branch, according to the instructor-set grading policy, based on items that have been passed and for which the learner's identity has been verified.

SQL create statement

CREATE TABLE course_branch_grades (
    course_branch_id varchar 
    ,ualberta_user_id VARCHAR(50) NOT NULL
    ,course_branch_grade_ts timestamp 
    ,course_passing_state_id int4 
    ,course_branch_grade_overall_passed_items int4 
    ,course_branch_grade_overall float4 
    ,course_branch_grade_verified_passed_items int4 
    ,course_branch_grade_verified float4 
    ,PRIMARY KEY (course_branch_id, ualberta_user_id, course_branch_grade_ts)
    ,FOREIGN KEY (course_branch_id) REFERENCES course_branches(course_branch_id)
    ,FOREIGN KEY (ualberta_user_id) REFERENCES users(ualberta_user_id)
    ,FOREIGN KEY (course_passing_state_id) REFERENCES course_passing_states(course_passing_state_id)
);
