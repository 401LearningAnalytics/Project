assessments

Description

Table of all the assessments / quizzes.

Columns

NameDescription
assessment_idEach quiz is assigned a unique id that is a combination of a base id and version.
assessment_base_idEvery quiz has a globally unique base id that is associated with a course.
assessment_versionWhenever a quiz is updated the base id is kept the same but the version and assessment id are updated.
assessment_type_idType of assessment so that it can be joined with the corresponding type specific tables.
assessment_update_tsUpdate timestamp for a change to the assessment.
assessment_passing_fractionNo column description available

SQL create statement

CREATE TABLE assessments (
    assessment_id VARCHAR(50) 
    ,assessment_base_id VARCHAR(50) 
    ,assessment_version INT4 
    ,assessment_type_id INT4 
    ,assessment_update_ts TIMESTAMP 
    ,assessment_passing_fraction FLOAT4 
    ,PRIMARY KEY (assessment_id)
    ,FOREIGN KEY (assessment_type_id) REFERENCES assessment_types(assessment_type_id)
);
