users

Description

Information about Coursera users

Columns

NameDescription
ualberta_user_idEncrypted Coursera user id for ualberta data.
user_join_tsThe timestamp of when the user joined Coursera.
country_cdThe country code of the user's location, based on IP address or as set by the user in their profile.
region_cdThe region code of the user's location, based on IP address or as set by the user in their profile.
profile_language_cdThe language code of the user's language, based on IP address or as set by the user in their profile.
browser_language_cdThe language code of the user's language, as set in their broswer.

SQL create statement

CREATE TABLE users (
    ualberta_user_id VARCHAR(50) NOT NULL
    ,user_join_ts timestamp 
    ,country_cd varchar 
    ,region_cd varchar 
    ,profile_language_cd varchar 
    ,browser_language_cd varchar 
    ,PRIMARY KEY (ualberta_user_id)
);
