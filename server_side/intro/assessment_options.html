assessment_options

Description

Information of choice options for a question

Columns

NameDescription
assessment_question_idunique question id of an assessment
assessment_option_idunique choice option ids of a question
assessment_option_displayThe JSON blob containing the question prompt. This is written in a custom Coursera markup language. The actual prompt text is encapsulated within key named "value."
assessment_option_feedbackThe text that is displayed to learners upon answering a question correctly
assessment_option_correctFor text answer type questions, instructors can insert both correct and incorrect answer patterns in order to provide learners with tailored feedback based on common solutions or misconceptions. This field indicates whether the pattern was one that corresponded to a correct solution (t) or an incorrect solution (f).

SQL create statement

CREATE TABLE assessment_options (
    assessment_question_id VARCHAR(50) 
    ,assessment_option_id VARCHAR(50) 
    ,assessment_option_display VARCHAR(20000) 
    ,assessment_option_feedback VARCHAR(20000) 
    ,assessment_option_correct BOOL 
    ,PRIMARY KEY (assessment_question_id, assessment_option_id)
    ,FOREIGN KEY (assessment_question_id) REFERENCES assessment_questions(assessment_question_id)
);
