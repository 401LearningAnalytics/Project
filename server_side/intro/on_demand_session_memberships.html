on_demand_session_memberships

Description

For each course's session, list the learners with the role of 'member' in that session and their start/end timestamps

Columns

NameDescription
course_id22 character unique string identifying an individual course
on_demand_session_idunique identifer for a course's session on the Phoenix platform
ualberta_user_idEncrypted Coursera user id for ualberta data.
on_demand_sessions_membership_start_tsthe datetime of when the learner became a 'member' role for this course's session
on_demand_sessions_membership_end_tsthe datetime of when the learner ended his or her 'member' role for this course's session

SQL create statement

CREATE TABLE on_demand_session_memberships (
    course_id varchar 
    ,on_demand_session_id varchar 
    ,ualberta_user_id VARCHAR(50) NOT NULL
    ,on_demand_sessions_membership_start_ts timestamp 
    ,on_demand_sessions_membership_end_ts timestamp 
);
