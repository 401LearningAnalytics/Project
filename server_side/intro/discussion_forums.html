discussion_forums

Description

For each course's discussion forums, contains the list of forums (e.g. General Discussions, Meet and Greet, etc.)

Columns

NameDescription
discussion_forum_idThe id of the discussion forum.
course_idCourse in which this forum exists
discussion_forum_titleTitle of the forum
discussion_forum_descriptionForum description
discussion_forum_orderThe order in which the discussion forum appears under the "Course Forums" label on the Discussions page.

SQL create statement

CREATE TABLE discussion_forums (
    discussion_forum_id VARCHAR(50) 
    ,course_id VARCHAR(50) 
    ,discussion_forum_title VARCHAR(20000) 
    ,discussion_forum_description VARCHAR(20000) 
    ,discussion_forum_order INT4 
    ,PRIMARY KEY (discussion_forum_id)
);
