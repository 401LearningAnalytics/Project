discussion_question_votes

Description

records when a discussion question was upvoted

Columns

NameDescription
ualberta_discussions_user_idEncrypted Coursera user id for ualberta discussions data.
course_id22 character unique string identifying an individual course
discussion_question_idThe id of the discussion question
discussion_question_vote_value0 = nothing, 1 = upvote
discussion_question_vote_tsTimestamp of vote

SQL create statement

CREATE TABLE discussion_question_votes (
    ualberta_discussions_user_id VARCHAR(50) NOT NULL
    ,course_id varchar 
    ,discussion_question_id varchar 
    ,discussion_question_vote_value int4 
    ,discussion_question_vote_ts timestamp 
    ,PRIMARY KEY (ualberta_discussions_user_id, discussion_question_id, discussion_question_vote_ts)
);
