peer_comments

Description

Peer comments on submissions

Columns

NameDescription
peer_comment_idprimary key
peer_submission_idThe submission that this comment is for
ualberta_peer_assignments_user_idEncrypted Coursera user id for ualberta peer_assignments data.
peer_comment_created_tsThe time when the comment was created. The comment never changes after creation
peer_comment_textThe text of the comment

SQL create statement

CREATE TABLE peer_comments (
    peer_comment_id VARCHAR(50) 
    ,peer_submission_id VARCHAR(50) 
    ,ualberta_peer_assignments_user_id VARCHAR(50) NOT NULL
    ,peer_comment_created_ts TIMESTAMP 
    ,peer_comment_text VARCHAR(65535) 
    ,PRIMARY KEY (peer_comment_id)
    ,FOREIGN KEY (peer_submission_id) REFERENCES peer_submissions(peer_submission_id)
);
