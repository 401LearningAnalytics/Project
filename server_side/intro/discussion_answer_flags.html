discussion_answer_flags

Description

records when a discussion answer is marked as resolved.

Columns

NameDescription
ualberta_discussions_user_idEncrypted Coursera user id for ualberta discussions data.
course_id22 character unique string identifying an individual course
discussion_answer_idThe id of an answer in the discussion forum.
discussion_answer_flag_activeA learner may flag a discussion answer, and later mark that answer as resolved. If a flag is still active, the value of "discussion_answer_flag_active" will be t. If the report was marked as resolved, the value will be f.
discussion_answer_flag_tsIndicates the time at which the discussion answer was flagged

SQL create statement

CREATE TABLE discussion_answer_flags (
    ualberta_discussions_user_id VARCHAR(50) NOT NULL
    ,course_id varchar 
    ,discussion_answer_id varchar 
    ,discussion_answer_flag_active bool 
    ,discussion_answer_flag_ts timestamp 
    ,PRIMARY KEY (ualberta_discussions_user_id, discussion_answer_id, discussion_answer_flag_ts)
);
