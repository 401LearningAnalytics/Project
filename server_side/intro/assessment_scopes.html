assessment_scopes

Description

Unique contexts in which the user can interact with the assessment

Columns

NameDescription
assessment_scope_idUnique contexts in which the user can interact with the assessment
assessment_scope_type_idthe id that links the assessmnt_scope_types table to see the assessment scope type description

SQL create statement

CREATE TABLE assessment_scopes (
    assessment_scope_id VARCHAR(50) 
    ,assessment_scope_type_id INT4 
    ,PRIMARY KEY (assessment_scope_id, assessment_scope_type_id)
    ,FOREIGN KEY (assessment_scope_type_id) REFERENCES assessment_scope_types(assessment_scope_type_id)
);
