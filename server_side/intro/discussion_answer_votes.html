discussion_answer_votes

Description

records when a discussion answer was upvoted or revoked.

Columns

NameDescription
ualberta_discussions_user_idEncrypted Coursera user id for ualberta discussions data.
course_id22 character unique string identifying an individual course
discussion_answer_idThe discussion answer to which the vote was applied
discussion_answer_vote_valueDiscussion answers can only be upvoted on the Phoenix platform; there is no such thing as a downvoting an answer. Upon upvoting, the value of this column is 1. However, if a user "revokes" their upvote, the value will instead be 0.
discussion_answer_vote_tsThe timestamp of when the vote was applied

SQL create statement

CREATE TABLE discussion_answer_votes (
    ualberta_discussions_user_id VARCHAR(50) NOT NULL
    ,course_id varchar 
    ,discussion_answer_id varchar 
    ,discussion_answer_vote_value int4 
    ,discussion_answer_vote_ts timestamp 
    ,PRIMARY KEY (ualberta_discussions_user_id, discussion_answer_id, discussion_answer_vote_ts)
);
