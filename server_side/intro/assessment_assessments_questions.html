assessment_assessments_questions

Description

Link table between the assessment and the questions data and describe high-level information of questions in an assessment.

Columns

NameDescription
assessment_idThe id that uniquely identifies an assessment
assessment_question_idThe id that uniquely identifies a question in an assessment
assessment_question_internal_idNo column description available
assessment_question_cuepointThe number of milliseconds into a lecture video that the in-video question is placed. For quizzes that are not in-video quizzes, it is 0.
assessment_question_orderOrder in which the questions are asked in the assessment.
assessment_question_weightNo column description available

SQL create statement

CREATE TABLE assessment_assessments_questions (
    assessment_id VARCHAR(50) 
    ,assessment_question_id VARCHAR(50) 
    ,assessment_question_internal_id VARCHAR(50) 
    ,assessment_question_cuepoint INT4 
    ,assessment_question_order INT4 
    ,assessment_question_weight INT4 
    ,PRIMARY KEY (assessment_id, assessment_question_id, assessment_question_internal_id)
    ,FOREIGN KEY (assessment_id) REFERENCES assessments(assessment_id)
    ,FOREIGN KEY (assessment_question_id) REFERENCES assessment_questions(assessment_question_id)
);
