discussion_question_flags

Description

records when an discussion question is flagged as resolved

Columns

NameDescription
ualberta_discussions_user_idEncrypted Coursera user id for ualberta discussions data.
course_id22 character unique string identifying an individual course
discussion_question_idThe id of the discussion question
discussion_question_flag_activewhether or not the flag is active
discussion_question_flag_tstimestamp of flagging

SQL create statement

CREATE TABLE discussion_question_flags (
    ualberta_discussions_user_id VARCHAR(50) NOT NULL
    ,course_id varchar 
    ,discussion_question_id varchar 
    ,discussion_question_flag_active bool 
    ,discussion_question_flag_ts timestamp 
    ,PRIMARY KEY (ualberta_discussions_user_id, discussion_question_id, discussion_question_flag_ts)
);
