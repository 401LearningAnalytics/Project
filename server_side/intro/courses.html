courses

Description

The list of Coursera's courses on the Phoenix platform, including info on its important dates (e.g. when it was launched)

Columns

NameDescription
course_id22 character unique string identifying an individual course
course_slugThe shortname of a course. This can be found in the course url, in the piece immediately following "coursera.org/learn/".
course_nameThe name of a course.
course_launch_tsThe timestamp of when a course was made available to learners.
course_update_tsThe timestamp of when a course was last updated.
course_deletedThe timestamp of when a course was deleted.
course_gradedIndicates whether a course contains graded items: 0 (no graded items), 1 (has graded items)
course_descThe course description which appears on the landing page
course_restrictedIndicates whether enroll access to the course is open or restricted only to eligible participants. All courses are unrestricted with the exception of deprecated courses: 0 (unrestricted), 1 (restricted)
course_verification_enabled_at_tsThe timestamp of when Course Certificates were enabled for a course.
primary_translation_equivalent_course_idThe course id for the primary translation of a course.
course_preenrollment_tsThe timestamp of when pre-enrollment was opened for a course.
course_workloadThe estimated workload of a course as shown on the course description page.
course_session_enabled_tsThe timestamp of when this course used sessions (instead of on-demand)

SQL create statement

CREATE TABLE courses (
    course_id VARCHAR(50) 
    ,course_slug VARCHAR(2000) 
    ,course_name VARCHAR(2000) 
    ,course_launch_ts TIMESTAMP 
    ,course_update_ts TIMESTAMP 
    ,course_deleted BOOL 
    ,course_graded BOOL 
    ,course_desc VARCHAR(10000) 
    ,course_restricted BOOL 
    ,course_verification_enabled_at_ts TIMESTAMP 
    ,primary_translation_equivalent_course_id VARCHAR(50) 
    ,course_preenrollment_ts TIMESTAMP 
    ,course_workload VARCHAR(100) 
    ,course_session_enabled_ts TIMESTAMP 
    ,PRIMARY KEY (course_id)
);
