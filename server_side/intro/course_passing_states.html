course_passing_states

Description

The descriptions of passing states for courses from the course_grades table

Columns

NameDescription
course_passing_state_ida unique identifier for course passing state
course_passing_state_desca description for course passing state (e.g. "passed", "verified passed")

SQL create statement

CREATE TABLE course_passing_states (
    course_passing_state_id INT4 
    ,course_passing_state_desc VARCHAR(255) 
    ,PRIMARY KEY (course_passing_state_id)
);
