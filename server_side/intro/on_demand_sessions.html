on_demand_sessions

Description

Contains sessions info (e.g start and end dates) and the Course Versioning used on the Phoenix platform

Columns

NameDescription
course_id22 character unique string identifying an individual course
on_demand_session_idunique identifer for a course's session on the Phoenix platform
on_demand_sessions_start_tsThe start date of a Phoenix session
on_demand_sessions_end_tsThe end date of a Phoenix session
on_demand_sessions_enrollment_end_tsThe end date of the enrollment period of a Phoenix session
course_branch_idEach session is associated with a single branch. Learners enrolled in a session see the course contents associated with this branch.

SQL create statement

CREATE TABLE on_demand_sessions (
    course_id varchar 
    ,on_demand_session_id varchar 
    ,on_demand_sessions_start_ts timestamp 
    ,on_demand_sessions_end_ts timestamp 
    ,on_demand_sessions_enrollment_end_ts timestamp 
    ,course_branch_id varchar 
);
