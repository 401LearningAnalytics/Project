feedback_item_ratings

Description

Contains data for item-level ratings

Columns

NameDescription
course_idCourse id for the course receiving feedback
course_item_idItem id for the item being rated
feedback_unit_idDetailed info for the element being rated. Will generally be a versioned id for item-level feedback
feedback_unit_typeThe type of the element being rated. Should be an item type
feedback_systemFeedback system: LIKE_OR_DISLIKE for item-level ratings
ualberta_feedback_user_idEncrypted Coursera user id for ualberta feedback data.
feedback_ratingValue of the rating. Should be 0 or 1 for item-level ratings
feedback_max_ratingMaximum value of the rating: 1 for item-level ratings
detailed_contextContains information on the element and its parents, usually including versioning information
feedback_tsTimestamp when the feedback was left

SQL create statement

CREATE TABLE feedback_item_ratings (
    course_id varchar 
    ,course_item_id varchar 
    ,feedback_unit_id varchar 
    ,feedback_unit_type varchar 
    ,feedback_system varchar 
    ,ualberta_feedback_user_id VARCHAR(50) NOT NULL
    ,feedback_rating int4 
    ,feedback_max_rating int4 
    ,detailed_context varchar 
    ,feedback_ts timestamp 
    ,PRIMARY KEY (course_id, course_item_id, feedback_unit_id, feedback_unit_type, feedback_system, ualberta_feedback_user_id, feedback_ts)
    ,FOREIGN KEY (course_id) REFERENCES courses(course_id)
    ,FOREIGN KEY (course_id, course_item_id) REFERENCES course_items(course_id, course_item_id)
);
