course_progress_state_types

Description

A mapping table from ids to descriptions of the different course progress states, such as 1 meaning "started".

Columns

NameDescription
course_progress_state_type_idThere are different possible progress state types, each state is given a number.
course_progress_state_type_descThe description of each possible progress state type. Currently 0 (not started), 1 (started), and 2 (completed).

SQL create statement

CREATE TABLE course_progress_state_types (
    course_progress_state_type_id int4 
    ,course_progress_state_type_desc varchar 
    ,PRIMARY KEY (course_progress_state_type_id)
);
