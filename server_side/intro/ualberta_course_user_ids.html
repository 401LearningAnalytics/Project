ualberta_course_user_ids

Description

Encrypted user id mapping table. Use this table to connect learner data from exports, which identifies ualberta learners using ualberta_user_id, to course-scoped data from external surveys.

Columns

NameDescription
ualberta_user_idEncrypted Coursera user id for ualberta data.
software_product_management_capstone_user_idEncrypted Coursera user id for software-product-management-capstone data.

SQL create statement

CREATE TABLE ualberta_course_user_ids (
    ualberta_user_id VARCHAR(50) NOT NULL
    ,software_product_management_capstone_user_id VARCHAR(50) NOT NULL
    ,PRIMARY KEY (ualberta_user_id)
    ,FOREIGN KEY (ualberta_user_id) REFERENCES users(ualberta_user_id)
);
