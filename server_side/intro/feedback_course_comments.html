feedback_course_comments

Description

Contains the contents of course reviews

Columns

NameDescription
course_idCourse id for the course receiving reviews
feedback_systemFeedback system: STAR for course reviews
ualberta_feedback_user_idEncrypted Coursera user id for ualberta feedback data.
feedback_categoryFor course reviews, the only category should be "generic"
feedback_textCML text of the review
feedback_tsTimestamp when the feedback was left

SQL create statement

CREATE TABLE feedback_course_comments (
    course_id varchar 
    ,feedback_system varchar 
    ,ualberta_feedback_user_id VARCHAR(50) NOT NULL
    ,feedback_category varchar 
    ,feedback_text varchar 
    ,feedback_ts timestamp 
    ,PRIMARY KEY (course_id, feedback_system, ualberta_feedback_user_id, feedback_category, feedback_ts)
    ,FOREIGN KEY (course_id) REFERENCES courses(course_id)
);
