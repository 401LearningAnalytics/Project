demographics_answers

Description

Stores answers to the questions from the demographics survey and user intent questions.

Columns

NameDescription
question_idThe indentifier for a particular question the learner answered. Can be used to connect demographics_answers, demographics_questions, and demographics_choices.
ualberta_demographics_user_idEncrypted Coursera user id for ualberta demographics data.
submission_tsThe timestamp for when the learner submitted their demographic survey.
choice_idThe identifier for the answer choice the learner selected for a particular question.
answer_intDemographics question answered by typing an interger (birth year, 4 digits).

SQL create statement

CREATE TABLE demographics_answers (
    question_id int4 
    ,ualberta_demographics_user_id VARCHAR(50) NOT NULL
    ,submission_ts timestamp 
    ,choice_id int4 
    ,answer_int int4 
);
