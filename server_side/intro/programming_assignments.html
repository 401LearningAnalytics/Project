programming_assignments

Description

Information of programming assignments

Columns

NameDescription
programming_assignment_idThis is a 'course_content_id', and is versioned. (e.g ABC@1 is version 1)
programming_assignment_base_idNone versioned id
programming_assignment_typeThe type ("standard", "graded", "closed").
programming_assignment_submission_typeThe type ("mulitpart")
programming_assignment_instruction_textThe CML Instruction content
programming_assignment_passing_fractionFraction of the maximum possible score that you must achieve in order to pass. This only exists for assignments that generate scores.
programming_assignment_submission_builder_schema_typeNo column description available
programming_assignment_submission_builder_schemaNo column description available
programming_assignment_update_tsWhen this version of the assignment was created.

SQL create statement

CREATE TABLE programming_assignments (
    programming_assignment_id VARCHAR(50) 
    ,programming_assignment_base_id VARCHAR(50) 
    ,programming_assignment_type VARCHAR(50) 
    ,programming_assignment_submission_type VARCHAR(50) 
    ,programming_assignment_instruction_text VARCHAR(65535) 
    ,programming_assignment_passing_fraction FLOAT8 
    ,programming_assignment_submission_builder_schema_type VARCHAR(50) 
    ,programming_assignment_submission_builder_schema VARCHAR(65535) 
    ,programming_assignment_update_ts TIMESTAMP 
    ,PRIMARY KEY (programming_assignment_id)
);
