quizData.php

Description:

This page is for fetching the data from database for quizes in the current course.

Given the course ID, this function fetches the quiz grade of different range, eg. 0% - 25%, 26%- 50%, 51% - 75%, 76% - 100%, for each componenet in the course and the number of students who fall into those ranges

Method:

Select COUNT(course_quiz_grade) from database and show on screen by quizUI.php

Parameters:

array $quiz_course_item_id : The array of componenet ID in the current course.

array $quiz_course_item : The array of names of componenet in the current course, with indices corresponding to the $quiz_course_item_id array.

array $quiz_count : The array of number of all finished quizes of each componenet, with indices corresponding to the $quiz_course_item_id array.

array $per_count : The array of number of students who fall into each range, the indices corresponds to the indices of components.
