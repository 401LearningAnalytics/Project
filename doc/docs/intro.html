Description: This page is scatter for Introduction to Software Product Management course. The scatter plot shows the current time versus performance for each student. Each dot in the plot means each student. The color means each student's process.

Method: Using open source d3 library in this file. This library provides the scatter plot which allows you to get data from database.

createBody()

        Create svg body of d3 library in javascripts

setDotAttr()

        Set up the attributes for each dots

setAxisAttr()

        Set up the attributes for x and y axis

Parameters:

parseTime: Get the time from database

formatTime: Format the showing time on chart

x: the x axis

y: the y axis

color: the color for dots
