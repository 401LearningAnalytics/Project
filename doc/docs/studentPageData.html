studentPageData.php

Description:

This page is for showing students performance in each component in all courses

It transfers from the course page to individual student page upon clicking on the dots which represent different students


Method:

fetchCourse()

       get course ID for course item grade table and show six charts which correspond to six different courses

fetchModule()

        get the module ID and names for each course, in which the chart will be created with multiple columns which are different componenets in a course

fetchScore()

        for each module in each course, fetch the corresponding score


Parameters:

int $student_id

        The id for current student.

array $courses

        The array of all courses the current student is in.

int $allmarks

        The marks of each component that belong to the student of all courses.

int $marks

        The marks of each component of individual courses.

int $row_names

        The names of the columns, which are the names of the components.
