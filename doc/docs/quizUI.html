quizUI.php

Description:

This page includes the javascript code, HTML code of generating quiz charts on the apge.

With given data, the page generates charts accordingly with the grades of quizes in each component


Method:

CreateQuizAverageChart()

       Insert paramater from PHP into JavaScipt to creat quiz info chart about the quiz average and standard deviation

CreateQuizSummaryChart()

        creat chart of quiz score percentage in each range for each module


Parameters:

str $name

        The names of components in this course for the chart columns.

float $per

        The marks of each quiz grade that belong to different rages(eg. 0% - 25%), in each component.

float $average

        The average of quiz grade for each component for current course.

float $standdev

        The standard deviation of quiz grade for each component for current course
