Way to test Test*.py

Have latest firefox browser installed
Have python 2.7 or higher 2 installed

Install seleniup API:
	http://selenium-python.readthedocs.io/installation.html

	sudo apt-get install python-pip
	sudo pip install selenium

Install WebDriver for Firefox:
	Install from https://github.com/mozilla/geckodriver/releases
	un-tar excutable
	add the filepath of the exe to PATH using:
		export PATH=$PATH:/directory exe is in


Run test file using
	python testfile.py
